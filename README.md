This uses the emotes /draw and /sheathe (appropriate to whether the weapon is drawn or sheathed) whenever the /drawers command is invoked.

This means you can make a macro calling /drawers and set it to your weapon draw/sheathe hotkey on a hotbar.

<img width="331" height="98" alt="image" src="https://github.com/user-attachments/assets/93d6a410-fa62-4327-bdf8-ba8d8be472cb" />

(The "/merror off" isn't needed, I'm just in the habit of using it.)

Alternatively, you can just use /drawers auto and it'll handle this itself.
